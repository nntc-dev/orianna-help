# Documentación del Sistema

Bienvenido a la documentación oficial del sistema ERP.

Este documento describe las funcionalidades principales, procesos de operación y referencias técnicas.

> Esta guía está dirigida a usuarios administrativos y personal de soporte.

## Acceso al Sistema

Para ingresar al sistema acceda a:

<https://erp.empresa.com>

1. Ingrese su usuario.
2. Capture su contraseña.
3. Presione **Iniciar sesión**.

> **Nota**
>
> Si tiene habilitada la autenticación de dos factores, deberá capturar el código generado por su aplicación autenticadora.

## Gestión de Empleados

El módulo de empleados permite administrar información laboral, contratos, puestos y movimientos de personal.

### Campos principales

| Campo      | Descripción                       |
| ---------- | --------------------------------- |
| EmployeeId | Identificador único del empleado. |
| FullName   | Nombre completo.                  |
| Status     | Estatus actual del empleado.      |

## Preguntas Frecuentes

### ¿Cómo recupero mi contraseña?

Utilice la opción **Olvidé mi contraseña** en la pantalla de inicio.

### ¿Dónde encuentro más documentación?

Consulte el portal interno de soporte o contacte al administrador del sistema.
